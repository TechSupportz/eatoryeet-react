A restaurant review website built for Sem 1.2 in Polytechnic

## Built using:

### Frontend
* React.js
* MUI
* Vite bundler
* React Redux Toolkit
* RTK Query

### Backend
* Express.js
* Node.js

### Database
* MySQL
